## Overview

This folder contains screenshots verifying the successful completion of the ASM1 lab.
The images document the process of connecting to the mainframe via SSH, assembling the source code, linking the object files, and verifying the output of the Fibonacci sequence program.

---

## Image Manifest

-   **01.png:** SSH connection to the IBM Z Xplore system and navigation to the `$HOME/assembly` directory.
-   **02.png:** Execution of the `as` command to compile `fibonacci.s` into the object file `fibonacci.o`.
-   **03.png:** Execution of the `ld` command to link the object file into the executable binary `fibonacci`.
-   **04.png:** Running the program with `./fibonacci` and viewing the output of the first 38 Fibonacci numbers.
-   **05.png:** Successful submission of the `CHKASM1` JCL job to claim credit for the challenge.
